# CODEX 00 - Verify Figma MCP

Verify Figma MCP availability without modifying files.

1. Check available Figma MCP tools.
2. Check available Figma skills.
3. Run the Figma account verification tool.
4. Confirm design-context tools such as `get_design_context`.
5. Confirm write-to-canvas tools such as `use_figma` and `create_new_file`.
6. Do not inspect random Figma files.
7. Do not modify any Figma file.
8. Record available tools, account result, and limitations.
