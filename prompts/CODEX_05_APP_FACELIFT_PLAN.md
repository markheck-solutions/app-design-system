# CODEX 05 - App Facelift Plan

Produce a facelift plan for one app without changing code.

Required output:

- App summary.
- Baseline findings.
- Selected layout pattern.
- Target Figma frame.
- Implementation phases.
- Affected files.
- Dependency risks.
- Accessibility considerations.
- Visual QA plan.
- Rollback plan.

Use `templates/APP_FACELIFT_PLAN.md`.
