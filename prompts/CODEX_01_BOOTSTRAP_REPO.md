# CODEX 01 - Bootstrap Repo

Bootstrap `C:\repos\app-design-system` as the shared design-system governance repo.

Rules:

- Stop if meaningful local files already exist.
- Initialize Git on `main`.
- Add remote `https://github.com/markheck-solutions/app-design-system.git`.
- Create required governance docs, tokens, prompts, templates, app registry, skills, and validation tools.
- Do not store secrets.
- Run `.\tools\validate-design-system.ps1`.
- Commit with `Initialize app design system governance repo`.
- Push to `origin main`.
