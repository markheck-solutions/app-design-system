# CODEX 02 - Create Figma Foundation

Create or update the `MS App Design System` Figma file using repo rules.

Before writing:

1. Read `AGENTS.md`.
2. Read `figma/FIGMA_FILE_STRUCTURE.md`.
3. Read `figma/FRAME_NAMING_CONVENTIONS.md`.
4. Read `docs/01_VISUAL_DIRECTION.md`.
5. Verify Figma account.

Create or verify the required pages and starter frames. Use working pages before approved pages. Do not overwrite approved frames without approval. Update `figma/FIGMA_LINKS.md` and `figma/MCP_USAGE_LOG.md`.
