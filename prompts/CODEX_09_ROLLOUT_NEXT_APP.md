# CODEX 09 - Rollout Next App

Repeat the facelift workflow for the next app.

Steps:

1. Pick app from `apps/registry.json`.
2. Confirm status and priority.
3. Complete app intake if missing.
4. Audit current UX.
5. Capture screenshots.
6. Import current state to Figma when available.
7. Create/select target frames.
8. Produce facelift plan.
9. Implement only after plan approval when required.
10. Run visual QA.
11. Update adoption tracking.
