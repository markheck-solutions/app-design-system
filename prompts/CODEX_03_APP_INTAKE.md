# CODEX 03 - App Intake

Collect and populate information for all 8 apps.

For each app, capture:

- App name.
- App id.
- Repo URL.
- Local path.
- Stack.
- Run command.
- Test command.
- Key routes.
- UI library.
- Styling system.
- Known UX problems.
- Target classification.
- Priority.
- Risks.

Update `apps/registry.json`, `docs/APP_ADOPTION_MATRIX.md`, and the matching `apps/app-XX/README.md`.
