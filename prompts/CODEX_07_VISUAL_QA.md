# CODEX 07 - Visual QA

Run visual QA against the design system and Figma.

Check:

- Target Figma frame linked.
- Baseline screenshots.
- After screenshots.
- Desktop layout.
- Responsive layout.
- Table readability.
- Status/severity consistency.
- Primary action clarity.
- Color contrast.
- Keyboard/focus behavior.
- Empty/loading/error states.
- Forbidden visual patterns.
- Backend behavior unchanged.
- Figma deviations documented.

Use `templates/VISUAL_QA_REPORT.md`.
