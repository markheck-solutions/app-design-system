# CODEX 06 - Implement From Figma

Implement from an approved Figma frame.

Rules:

- Read this repo's `AGENTS.md` and required docs first.
- Inspect the target app stack before changes.
- Use approved Figma frame links.
- Preserve business logic and data contracts.
- Do not add UI dependencies without approval.
- Make small, reviewable changes.
- Capture before/after screenshots.
- Run app checks and visual QA.
- Update adoption tracking.
