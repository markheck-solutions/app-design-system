# Figma Handoff

Purpose: use Figma frame links and MCP design context safely.

Steps:

1. Verify MCP account.
2. Use selected frame links.
3. Batch design-context reads when possible.
4. Do not overwrite approved frames.
5. Write only to working pages unless approved.
6. Log major MCP actions.
7. Document Figma-to-code deviations.

Stop and ask when:

- Account appears wrong.
- Requested write would modify existing approved designs.
- A node-specific frame link is missing for implementation work.
