# App Rollout Workflow

1. Bootstrap design-system repo.
2. Create Figma foundation.
3. Register all 8 apps.
4. Pick pilot app.
5. Import pilot current state to Figma.
6. Create 3 target directions.
7. Select approved direction.
8. Implement pilot.
9. QA pilot.
10. Extract reusable lessons into design system.
11. Roll out to apps 2 through 8.

## Pilot Selection Criteria

- High visibility.
- Currently weak UI.
- Moderate implementation risk.
- Representative layout value.
- Business value.
- Clear workflow.
- Low backend risk.
