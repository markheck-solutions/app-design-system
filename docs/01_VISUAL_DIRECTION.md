# Visual Direction

Name: Enterprise Operations Command Center

## Intended Feel

- Precise.
- Modern.
- Operational.
- Trustworthy.
- Executive-ready.
- High-signal.

## Approved Traits

- Strong hierarchy.
- Restrained color.
- Deliberate elevation.
- Clear status/severity treatment.
- Dense but readable layouts.
- Action-oriented workflows.

## Rejected Traits

- Random gradients.
- Generic AI dashboard look.
- Toy SaaS styling.
- Unnecessary glassmorphism.
- Oversized decorative icons.
- Decorative charts that do not support decisions.
- Excessive whitespace.
- Weak contrast.
- Inconsistent statuses.
