# Visual QA

Use this checklist before calling facelift work complete.

- [ ] Target Figma frame linked.
- [ ] Baseline screenshots captured.
- [ ] After screenshots captured.
- [ ] Desktop layout checked.
- [ ] Responsive layout checked.
- [ ] Table readability checked.
- [ ] Status/severity consistency checked.
- [ ] Primary action clarity checked.
- [ ] Color contrast checked.
- [ ] Keyboard/focus behavior checked.
- [ ] Empty/loading/error states checked.
- [ ] No forbidden visual patterns.
- [ ] No backend behavior changes.
- [ ] Deviations from Figma documented.
