# Design Tokens

Tokens live in `tokens/design-tokens.json` and `tokens/css/theme.css`.

The current token set is portable and framework-agnostic. It covers:

- Color roles.
- Background.
- Surface.
- Border.
- Text.
- Muted text.
- Accent.
- Severity/status colors.
- Spacing scale.
- Radius scale.
- Typography scale.
- Elevation scale.
- Layout widths.
- Motion timing.

## Status And Severity Roles

- neutral
- info
- success
- warning
- critical
- blocked

Do not invent app-specific status colors. Extend labels first, not color roles.
