# MCP Usage Policy

- Use Figma MCP deliberately.
- Prefer selected frame links.
- Batch design-context reads when possible.
- Use write-to-canvas only on working/draft pages unless explicitly approved.
- Do not overwrite approved pages without approval.
- Log major MCP actions in `figma/MCP_USAGE_LOG.md`.
- Stop and ask if the Figma account appears wrong.
- Stop and ask if write-to-canvas would modify existing approved designs.
