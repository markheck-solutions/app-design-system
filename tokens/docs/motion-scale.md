# Motion Scale

Motion should clarify state change and preserve operational confidence.

| Token | Value | Use |
|---|---:|---|
| fast | 120ms | hover/focus micro interactions |
| base | 180ms | menus, drawers, common transitions |
| slow | 260ms | larger panels or route-level movement |
| ease | cubic-bezier(0.2, 0, 0, 1) | standard easing |

Respect reduced-motion preferences where supported.
