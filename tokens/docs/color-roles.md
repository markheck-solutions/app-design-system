# Color Roles

The palette is restrained and operational. Use color to clarify state, severity, and interaction priority.

Core roles:

- `background`: app canvas.
- `surface`: normal content surfaces.
- `surfaceRaised`: elevated drawers, popovers, and command panels.
- `surfaceSubtle`: subdued panels and table header areas.
- `border`: default divider and input border.
- `borderStrong`: emphasized divider or selected state.
- `text`: primary text.
- `textMuted`: secondary text.
- `textSubtle`: tertiary metadata.
- `accent`: primary actions and selected navigation.

Status roles:

- `neutral`: normal pending or inactive.
- `info`: active informational state.
- `success`: verified positive completion.
- `warning`: risk but not blocked.
- `critical`: urgent or severe failure.
- `blocked`: progress cannot continue.
