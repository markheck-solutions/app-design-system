# Spacing Scale

Use the spacing scale for layout rhythm, gaps, padding, and density.

| Token | Value | Use |
|---|---:|---|
| 0 | 0px | reset |
| 1 | 4px | tight inline gaps |
| 2 | 8px | compact control gaps |
| 3 | 12px | table/cell rhythm |
| 4 | 16px | standard panel padding |
| 5 | 20px | medium section gaps |
| 6 | 24px | page groups |
| 8 | 32px | major sections |
| 10 | 40px | wide section separation |
| 12 | 48px | page-level spacing |
