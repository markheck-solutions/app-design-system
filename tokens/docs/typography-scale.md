# Typography Scale

Use typography to support scanning and hierarchy. Avoid hero-scale type inside dashboards, tables, forms, or tool surfaces.

| Role | Size | Line height | Weight | Use |
|---|---:|---:|---:|---|
| display | 32px | 40px | 700 | rare landing/cover/title use |
| heading | 24px | 32px | 700 | page title |
| section | 18px | 26px | 650 | panel/section title |
| body | 14px | 20px | 400 | default content |
| small | 12px | 16px | 400 | metadata and compact labels |

Default family: Inter, Segoe UI, Arial, sans-serif.
